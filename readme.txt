Displays user ads on their own articles.

== Description ==

A really simple plugin that allow users from a settable minimum level to add some text before and after their posts, this can be their adsense code or anything else.

Admin can set a default text, something to add before and after user text and the minimum level users can use this plugin.

Can also be useful for a multi user blog where you want to allow signatures as well as an adsense or anything else plugin for your blog.
